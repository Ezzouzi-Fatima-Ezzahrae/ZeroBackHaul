# 🚚 SmartTruck AI

AI-powered logistics platform to detect empty trucks and optimize fleet usage.

## Features
- Detect empty trucks in real time
- Interactive dashboard with KPIs and charts
- Map visualization of fleet locations
- AI-powered insights using Google Gemini
- Chat assistant for logistics decisions

## Tech Stack
- Python
- Streamlit
- Plotly
- Google Gemini API

## ▶️ Run the app
```bash
streamlit run EmptyMiles.py
